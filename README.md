Introduction
Digital Dreams is an IT ecommerce company that provides a wide range of services, including web development, mobile app development, and custom software solutions. This landing page is designed to showcase the company's expertise, services, and portfolio, as well as provide a way for potential clients to get in touch.
Features
Responsive Design: The landing page is designed to be mobile-friendly and responsive, ensuring a great user experience across different devices.
Smooth Animations: The portfolio section and testimonials section feature smooth animations and hover effects, making the page more visually appealing.
Contact Form: The landing page includes a contact form that allows users to get in touch with the company.
Google Map Integration: The landing page includes a Google Map that shows the company's location, making it easier for users to find the business.
Components
The landing page consists of the following components:
Navbar: The navigation bar at the top of the page, which includes links to different sections of the site.
Hero Section: The main hero section of the landing page, which includes a brief introduction to the company and a call-to-action button.
Services Section: A section that showcases the company's various IT ecommerce services.
Why Choose Us: A section that highlights the company's unique selling points and why customers should choose them.
Portfolio Section: A section that showcases the company's past projects and work.
Testimonials Section: A section that features testimonials from satisfied clients.
Contact Form: A section that includes a contact form for users to get in touch with the company.
Google Map: A section that includes a Google Map showing the company's location.
Footer: The footer of the landing page, which includes company information, links to various pages, and social media links.
Getting Started
To get started with the Digital Dreams landing page, follow these steps:
Clone the repository:
bash
git clone https://github.com/your-username/digitaldreams.git

Install the dependencies:
bash
cd digitaldreams
npm install
# or
yarn install

Run the development server:
bash
npm run dev
# or
yarn dev

Open your web browser and navigate to http://localhost:3000 to see the landing page.
Deployment
The Digital Dreams landing page is deployed on Vercel, a popular hosting platform for Next.js applications. You can access the live demo at digitaldreams-p51ofuuca-samritbasnets-projects.vercel.app.
To deploy the landing page on your own, you can follow the Next.js deployment documentation.
